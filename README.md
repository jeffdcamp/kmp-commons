# Kmp-Commons
Kotlin Multiplatform shared/common code library for Android/iOS/Desktop apps and libraries

## Features

Helpers for

* Analytics
* Datastore
* Ktor
    * DirectDownloader
    * ApiResponse
    * CacheApiResponse
* Function extensions for
    * Enum
    * KotlinDateTime
    * Ktor
    * Okio
    * Flow
    * Map
* Logging
* Text
    * Csv Export
    * EmailUtil
    * NumberFormatExt
